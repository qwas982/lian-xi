# 我设想一种使用布尔运算的"字节门神经网".    

它不是说非得基于一字节或字节的倍数, 也可以是若干位bit作为基本元素.    

它是原生并行的,    
我把它建立在以下需求上:    
字节门 + 布尔运算 + 纯整数 + 幂塔 + 离散微积分.    
完全不使用浮点, 避免高功耗、高成本.    
字节门神经网一样是基于深度学习.    

# 主要逻辑如下;    
由海量字节门在布尔运算的支持下, 产生海量状态,    
使用这些状态构造流形,    
在流形上作张量计算,    
当然也可以做流形计算, 例如坐标卡 测地线 曲率等等.    
这种流形层递归多层,    
每层可用超平面提取表征, 并投影/映射到下一层流形,    
字节门怎么连, 布尔运算怎么算, 由流形的构造需求决定,    
使用裂变遍历与聚变遍历在每层做感知计算, 多层也可以,    
使用高斯泼溅 + 光追渲染进行可视化调试,    
传感器信号/数据不进入字节门 + 布尔运算层,    
而是从流形层进入,    
我把文本 图片 音频 视频 五觉六感 电磁波等等, 看做传感器.    
因为流形层可做张量计算, 所以传统深度学习所用的那些技术,    
例如transformer 注意力机制 扩散 卷积 JEPA等等方法都可移植,    
只不过基于浮点的连续微积分不能用了, 所以要进行适应型改良,    
至于如何学习,    
字节门神经网有基于自身特点的原生学习方式, 它无须训练,    
不过也可以使用反向传播或向前向前_FF算法,    
使用纯整数, 是因为考虑到通用与效率, 低能耗 低成本,    
使用幂塔可以获得海量数值, 提供的范围比浮点要高,    
且要比浮点更精确与确定型,    
可以用纯整数做离散微分几何计算,    
而离散微积分可以做近似计算, 例如像素游戏它也可以看起来是连续的,    
只要分辨率够高, 就可以做到近似, 但没必要,    
我们需要的是智能, 而不是极致地工程炫技,    
基于离散 + 纯整数, 一样可以获得机器智能, 且也是数字式的,    

以上描述,    
主要是解决目前大模型技术没有真智能, 高功耗高成本, 无法感知物理世界,    
无法获得物理智能的问题.    

人类的一切知识STEMA, 皆是基于对物理 对物质 对空间的感知, 才产生,    
而一切现实是建立在流形上, 爱因斯坦说时间不存在.    
所以我的出发点, 是构造一个模拟的流形, 让智能自动演化出来,    
通过递归的流形分层, 来创造抽象, 内部原生并行感知,    
所以, 它的思考就是多层流形的活动, 而意识与心智, 或许可以从规模里涌现,    

既然高维张量涌现出了低维流形, 那为啥不直接从流形下手呢,    
兜兜转转在连续张量上做关系统计, 不是一个敏捷的做法.    

目前的大模型技术实现的AI, 是在一个人类给它的数据上进行统计内部关联,     
并投影到一个高维张量上, 看起来像是那么回事(构造了一个流形),    
这也是大模型为啥会有幻觉、不懂物理、没有常识的原因,    
仔细推理, 人类给它的数据是怎么来的?    
是人类这个碳基神经网, 从现实流形里抽象出来的产物, 例如文本图片音频视频,    
大模型获得的数据源, 并非来自于物理世界这个一手源泉,    
真正的智能, 显然不可能从二手数据里找关系并放大关系产生,    

而现在大模型技术所用的找关系并放大关系, 能否用于物理流形的现实世界,     
目前, 我没看到有相关应用的,    
但是, 这种技术丢失了很多物理流形的信息量(熵)是可以确定的,    
本来经过人类翻译一遍的数据, 就已经丢失了很多至关重要的信息,    
所以, 人类应该尽可能让AI直接接触物理实在,    
人类只需要创造出能容纳这个投影的容器即可,     
智能或许没有多复杂, 它只需要一个环境与初始条件,
