### 3.2 数字干细胞设计
每个数字干细胞是一个抽象图灵机，包含：
- **状态寄存器**：存储当前分化状态（如未分化、神经祖细胞、肌肉细胞等）。
- **环境输入带**：接收数字培养皿的信号（损伤标记、营养浓度、细胞间信号）。
- **操作输出带**：执行动作（分裂、分化、分泌信号、凋亡）。
- **归约引擎**：根据当前状态和输入，从数字基因组中匹配并执行基因λ项，更新状态并输出操作。

### 3.3 数字基因组
数字基因组是一组用λ演算编写的函数库。每个基因λ项接受环境参数，返回控制信号。例如：
- `(λ损伤. 分裂)`：若检测到损伤信号，执行分裂。
- `(λ[Notch>阈值]. 分化神经)`：若相邻细胞释放的Notch信号超过阈值，分化为神经细胞。

### 3.4 数字培养皿
简化环境模拟：仅包含损伤标记（二进制）、营养梯度（标量场）、信号扩散（简化扩散方程）。复杂度由硬件性能限定。

### 3.5 生长过程
从单个数字干细胞启动，在培养皿中接收到损伤信号后，自动触发归约、分裂、分化、迁移等行为，最终长出复杂的组织结构（如神经网络）。整个过程中，没有显式编程如何再生，全部行为从基因λ项的归约中涌现。

|---------|-------------|---------|
| 注意力 | 测地线注意力 | 计算当前状态到其他点的测地距离，距离近的获得高权重 |
| 记忆 | 动态吸引子网络 | 记忆作为流形上的吸引子，当前状态沿流形演化落入最近吸引子 |
| 学习 | 局部预测代码 | 每个节点预测邻居，误差驱动局部更新；赫布可塑性增强同时活跃的连接 |
| 决策 | 主动推理 | 内部目标对应吸引子盆地，沿流形梯度向目标移动 |
| 工作记忆 | 短时轨迹缓冲 | 当前轨迹片段暂存于工作记忆流形 |

### 4.4 流形计算的优势
- **连续时间**：使用神经常微分方程描述状态演化，而非离散时间步。
- **局部性**：全部计算仅依赖局部邻居，避免全局反向传播。
- **低功耗**：流形计算天然稀疏，符合生物高效性。

## 5. 两框架的融合：从生长到运作的完整生命周期

数字再生提供**结构的诞生**，流形计算提供**结构的运作**，两者通过以下接口无缝衔接：

```
[数字再生层]                      [流形计算层]
     |                                  ↑
     | 生长出递归图结构                  | 在结构上演化
     ↓                                  |
[初始神经网络] ---- 环境交互塑造流形 ---→ [动态流形]
     |                                  |
     | 损伤/需求触发再生                  | 认知需求反馈
     ↓                                  |
[再生响应] ←────────── 反馈 ────────────┘
```

### 5.1 关键接口：神经发生
当数字干细胞分化为神经元时，执行“轴突生长基因”，根据环境化学梯度和电活动决定连接目标。突触形成后，连接强度受局部活动调节（赫布学习）。这保证了生长出的网络具备初始功能结构。

### 5.2 双向调节
- **向下**：流形计算层的持续活动会影响基因表达（如通过钙信号调控转录），实现“用进废退”的发育可塑性。
- **向上**：数字再生层在检测到结构损伤或功能不足时，启动局部干细胞增殖分化，修复或扩展网络。

## 7. 过程演示：从单个干细胞到简单数字智能

以下演示一个简化案例：数字再生长出能趋利避害的微型神经系统。

### 阶段1：启动
- **输入**：数字培养皿中放置一个数字干细胞，设定初始状态为“全能”。
- **基因组**：包含基础基因——分裂基因（营养充足时）、趋化基因（向营养梯度移动）、伤害响应基因（接触毒素时凋亡并释放警告信号）。

### 阶段2：增殖与分化
- 营养充足 → 分裂基因激活 → 产生4个子细胞。
- 局部细胞密度达到阈值 → 分化基因激活 → 其中两个分化为感觉神经元（表达毒素受体），两个分化为运动神经元（控制纤毛摆动）。
- 轴突生长基因引导神经元连接：感觉神经元与运动神经元建立突触。

### 阶段3：流形形成
- 4个神经元形成简单递归图，其活动空间为4维，但受环境约束自然落于2维流形（一个维度对应营养梯度，一个维度对应毒素浓度）。
- 吸引子形成：高营养区对应一个稳定点，低营养区对应另一个。

### 阶段4：行为涌现
- 系统在培养皿中移动：感觉神经元检测到营养梯度，通过突触驱动运动神经元，使纤毛向营养源摆动。
- 若遇到毒素，感觉神经元激活伤害响应基因，释放信号使周围细胞逃离，同时自身凋亡。剩余干细胞感知到凋亡信号，启动再生，补充神经元。

### 阶段5：持续学习
- 若多次在特定方向遇到毒素，流形上形成新的吸引子（“危险方向”），系统学会提前避开。
- 整个过程无需预训练，全凭基因规则和环境交互自发涌现。

## 8. 讨论：5亿参数能否实现狗级智能？

狗脑有约5亿神经元，但突触连接达数千亿。在我们的框架中：
- **5亿参数用于存储生长规则和基本特性**：数字基因组（5000万）、干细胞运行时（1亿）、调控网络（5000万）、局部学习规则（5000万）、环境接口（5000万）、剩余动态分配（1亿）。
- **最终神经网络连接数**：通过生长过程动态创建，可达数千亿，受硬件内存限制但可扩展。
- **关键优势**：知识存储在连接中，而非参数中；世界作为外部存储；连续动力学提升效率。

因此，**5亿参数 + 数字再生 + 流形计算 + 持续环境交互，理论上足以支撑狗级通用智能**。这解释了生物为何能以极低功耗实现强智能。
